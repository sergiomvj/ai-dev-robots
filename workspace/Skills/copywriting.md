# ✍️ SKILL: Copywriting Persuasivo
> **Cria textos de marketing, anúncios e conteúdo otimizados para conversão.**

---

## Parâmetros

- **Objetivo** — venda, engajamento, awareness, retenção
- **Tom** — profissional, casual, urgente, inspiracional
- **Canal** — Facebook Ads, Instagram, email, landing page
- **Público-alvo** — descrição do ICP (Ideal Customer Profile)

## Frameworks suportados

| Framework | Uso ideal |
|---|---|
| AIDA | Anúncios e emails frios |
| PAS | Problemas e dores do cliente |
| FAB | Features de produtos |
| Storytelling | Conteúdo de marca |

## Como usar

```
Prompt base:
"Use o framework AIDA para criar um anúncio Facebook Ads 
para [PRODUTO] direcionado a [PÚBLICO], tom [TOM], 
objetivo [OBJETIVO]. Máximo 150 palavras."
```

## Checklist de qualidade
- [ ] Headline com benefício claro (não feature)
- [ ] CTA único e específico
- [ ] Prova social ou credibilidade incluída
- [ ] Sem jargões técnicos para o público-alvo
- [ ] Urgência ou escassez quando aplicável
